Добавляет к Woocommerce возможность отправлять SMS с использованием своего Android телефона

== Description ==

**SMS Gateway** для  Woocomerce позволяет отправлять SMS c телефона при добавлении нового заказа и об изменении статуса заказов. Для отправки сообщений необходимо зарегистрироваться на [htmlweb.ru](https://htmlweb.ru/user/signup.php?utm_source=WPplug "Регистрация") и загрузить приложение [SMS Шлюз](https://play.google.com/store/apps/details?id=ru.sp2all.smsgate) на телефон.

Настроить оповещения для разных групп пользователей можно для стандартных статусов заказов [Woocommerce](https://ru.wordpress.org/plugins/woocommerce/).
* Новый заказ
* В ожидании оплаты
* Обработка
* На удержании
* Выполнен
* Отменен
* Возвращен
* Не удался

Плагин также позволяет создать рассылку sms группам пользователей.

Автор: oruskade@gmail.com

== Installation ==

1. Устиновите плагин через страницу плагинов WordPress или загрузите вручную файлы плагина в каталог `/wp-content/plugins/sms-gateway`.
1. Активируйте плагин в разделе "Плагины" в консоли Wordpress
1. Задайте необходимые настройки в разделе "SMS Gateway" в консоли Wordpress. Также можно перейти к настойкам плагина, нажав "Настройки" в разделе "Плаагины" в консоли Wordpress под названием плагина.
1. Для отправки сообщений необходимо в настройках указать API ключ, узнать который можно в [личном кабинете](http://htmlweb.ru/user/) на [htmlweb.ru](http://htmlweb.ru/) после регистрации.

== Frequently Asked Questions ==

= Как узнать API ключ? =
 
Для получения API ключа необходима регистрация на [htmlweb.ru](http://htmlweb.ru/?utm_source=WPplug). API ключ можно узнать в [личном кабинете](http://htmlweb.ru/user/?utm_source=WPplug) на сайте.
 
= Как отправлять сообщения? =
Для отправки сообщений необходимо установить приложение SMS Шлюз на свой телефон. 
Чтобы совершить **рассылку**, необходимо на странице настроек плагина на вкладке "Рассылка" ввести текст сообщения и выбрать группу, для которой требуется сделать рассылку.  
Чтобы настроить автоматическую отправку **оповещений**, на вкладке "Оповещения" для конкретного события  необходимо выбрать получателей, задать текст  sms, отметить оповещения как активное и сохранить настройки с помощью кнопки "Сохранить".
 
= Как просмотреть отправленные sms? =
Отправленные сообщения и их статусы можно узнать на вкладке "Отправленные"
